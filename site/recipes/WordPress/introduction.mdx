## Install core packages

```shell
yarn add @roots/bud @roots/bud-cli --dev
```

## Install React

```shell
yarn add @roots/bud-babel @roots/bud-react --dev
```

## Install WP config presets

```shell
yarn add @roots/bud-wordpress-dependencies @roots/bud-wordpress-externals @roots/bud-wordpress-manifests @roots/bud-entrypoints --dev
```
