# Configuring React

This guide covers adding support for React to your project.

## Installing extensions

Install the **@roots/bud-babel** and **@roots/bud-react** extensions.

<Tabs
  defaultValue="yarn"
  values={[
    {label: 'yarn', value: 'yarn'},
    {label: 'npm', value: 'npm'},
  ]}>
  <TabItem value="yarn">

    yarn add @roots/bud-babel @roots/bud-react --dev

  </TabItem>
  <TabItem value="npm">

    npm install @roots/bud-babel @roots/bud-react --save-dev

  </TabItem>
</Tabs>

Remember to run **bud init** after installation.

## Base configuration

Add both extensions to your configuration with [**bud.use**](/docs/bud.use). 

React depends on Babel, so ensure that [**@roots/bud-babel**](/extensions/bud-babel) is included first.

```js title='bud.config.js'
const babel = require('@roots/bud-babel')
const react = require('@roots/bud-react')

module.exports = app => 
  app.use([babel, react])
```
